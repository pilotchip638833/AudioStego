# AudioStego
An audio steganography project exploring bit manipulation and how hidden data can be embedded within audio files.

# The files

encode.html : a minimal encode-only version
-
decode.html : a minimal decode-only version
-
AudioStego : a full working version including complete UI

# How to use

Download the files
------------------------------------------------------------------------------------------------------------------
Open with any modern browser
------------------------------------------------------------------------------------------------------------------
No installations or server required
------------------------------------------------------------------------------------------------------------------

# Known limitations

16-bit WAV files only
File must not be re-compressed after encoding or the message will be lost

Built with
-
HTML, CSS, JavaScript — no external libraries.
